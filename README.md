# GVK

Hobby Vulkan renderer written in C++, with the Khronos Vulkan Tutorial as a reference.

## Progress

### Core
- [x] Vulkan initialization
- [x] Presentation
- [x] Graphics pipeline
- [x] First triangle
- [x] Vertex and index buffers
- [x] Uniform buffers
- [x] Descriptor sets
- [x] Texture loading and sampling
- [x] Depth buffering
- [x] OBJ model loading
- [x] Basic engine modularization (`GVKCommon`, `GVKAsset`, `GVKRender`)

### Planned
- [ ] Support arbitrary models (e.g. glTF)
- [ ] Material system
- [ ] PBR lighting
